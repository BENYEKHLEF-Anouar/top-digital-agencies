Permalinks & URL Structure

To maximize SEO performance and ensure easy user navigation, we define the following rewrite rules and permalink mappings. French is the default root; English is prefixed with /en/.

Content Type Purpose French (Default Root) English (Prefixed)
Home Page Root Homepage Route / /en/
Static Pages Standard Core Pages /a-propos//contact//methodologie/ /en/about//en/contact//en/methodology/
Directory Main Search & Filter Interface /annuaire/ /en/directory/
Rankings Curated Editorial Leaderboard /classements/ /en/rankings/
Agency CPT Singular Agency Profile Details /agence/%postname%/ /en/agency/%postname%/
Specialty Hub CPT Search-Engine Optimized Landings /services/%postname%/ /en/services/%postname%/
Blog Posts Editorial Articles / Guides /blog/%postname%/ /en/blog/%postname%/
Service Taxonomy Agencies by Specific Channel /service/%term_slug%/ /en/service/%term_slug%/
City Taxonomy Agencies by Geographic Location /ville/%term_slug%/ /en/city/%term_slug%/

03

Schema Fields & Attributes

Data relationships between our Custom Post Types, Taxonomies, and Custom Fields (ACF).

Agency agency CPT

Standard Fields: Title, Excerpt (descriptions), Featured Image (custom logo).

Taxonomies: agency_service, agency_city.

  • logo_textText — Initials logo fallback (e.g. RMD).
  • logo_imageImage — Vetted custom image logo file.
  • rating_valueText — Lighthouse average rating score (e.g. 4.9).
  • review_countText — Count of verified reviews (e.g. 42).
  • agency_rankText — Leaderboard rank position (e.g. 1).
  • budgetText — Minimum monthly project budget (e.g. 15,000 MAD/mo).
  • projectText — Average campaign project scope (e.g. 50K-200K MAD).
  • team_sizeText — Count of team employees (e.g. 25-50).
  • clients_servedText — Historic customer totals (e.g. 150+).
  • foundedText — Year established (e.g. 2018).
  • addressText — Local office address details.
  • emailEmail — Contact email address.
  • phoneText — Contact phone number.
  • websiteText — Company website redirection link.
  • pagespeed_scoreNumber — Technical Lighthouse speed index (0–100).
  • core_web_vitalsSelect — PASS / FAIL performance status.
  • code_cleanlinessNumber — Performance cleanliness index percentage.
  • case_studiesRepeater — Vetted portfolio case listings:
    • titleText — Case study headline.
    • tagText — Optimization channel / taxonomy category.
  • why_listedRepeater — Points explaining why the agency was chosen:
    • point_textText — Detailed criteria check.
Specialty Hub specialty_hub CPT
  • icon_svgTextarea — Raw XML markup for Lucide Icon tags.
  • direct_link_parameterText — Target taxonomy slug filter map (e.g. seo).
  • sub_servicesRepeater — Checklist items:
    • service_nameText — Service bullet.
Stat Metric stat_metric CPT
  • stat_numberText — Vitals count (e.g. 150+).
  • stat_labelText — Metric description label.
Partner Logo partner_logo CPT

Standard Fields: Title (Company name).

Featured Image: File upload for partner logo image.

Blog Post post Post Type

Standard Fields: Title, Content, Featured Image, Author, Date.

Taxonomies: Category (Ranking, Guide, Comparison).

  • embedded_listingsRelationship — Linked CPT agency posts featured inside the article layout.

06

Dynamic Routing & Filter Mappings

In the mockup, location and channel filtering use JavaScript query parameters. In the final WordPress theme these translate directly into dynamic taxonomy queries.

Service Filters

e.g. service=SEO

Maps to the custom taxonomy agency_service. Entering /service/seo/ automatically displays the pre-filtered SEO agency listings using the standard taxonomy template taxonomy-agency_service.php (reusing the directory design layout).

City Filters

e.g. city=Casablanca

Maps to the custom taxonomy agency_city. Entering /ville/casablanca/ displays the Casablanca pre-filtered list using taxonomy-agency_city.php.
